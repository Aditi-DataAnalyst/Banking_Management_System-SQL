Bank Management System (Oracle SQL Developer)

Project Overview
Developed a Bank Management System using "Oracle SQL Developer" for database design, management, and SQL query execution.

Database Design
- Tables: CUSTOMER, ACCOUNTS, TRANSACTIONS
- Features: Primary keys, foreign keys, constraints
- SQL Operations: CRUD (Create, Read, Update, Delete), joins,grouping,indexing,Ranking,View 

 SQL Highlights
- Created normalized tables with Oracle datatypes (`VARCHAR2`, `NUMBER`, `CHAR`)
- Implemented constraints for data integrity (PRIMARY KEY, NOT NULL)
- Performed queries for:
  - Customer demographics (city, gender)
  - Account balances (average, highest, lowest)
  - Transaction analysis (deposits, withdrawals, transfers)
- Used Oracle SQL commands like `DESC`, `SELECT`, `JOIN`, `GROUP BY`

Tools Used
- Oracle SQL Developer → Database creation, schema design, query execution

 Key Outcomes
- Built a functional banking database with realistic customer, account, and transaction data
- Demonstrated SQL skills in table creation, joins, grouping, and constraints
- Recruiter‑friendly project showcasing Oracle SQL Developer expertise
